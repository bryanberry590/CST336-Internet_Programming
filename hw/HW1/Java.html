About Java

Java is a very common language and is probably one of the first coding languages people hear about. Java was originally developed by a man named James Gosling back in 1995 as said by javatpoint. Java is an object-oriented programming language and is categorized as a high level language due to its functionality and readability. Java is commonly used in a multitude of computer science industries. These industries include web application, android development, video games, cloud applications, and many more. Java is a great starting language due to how user friendly it is at the start. A good foundation in java will go a long way when learning future languages.
